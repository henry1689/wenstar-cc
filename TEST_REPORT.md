## 八、已知限制与待改进

1. **M3 Arousal 检测偏保守** — 无感叹号/emoji 时 arousal 偏低，当前为设计取舍（宁可低估不高估）
2. **M5 当前使用 MockLLMProvider** — 模板填充式回应，接入真实 LLM 后表达力会大幅提升
3. **M5 积极分享场景** — calcium 0.53 在 level 1 区间只能产出 memorize，要产出 ask 需 level 2（钙质≥0.6）或接入 LLM 后由 Prompt 自主决定
4. **家族图谱** — 当前只支持 infer 亲属关系 + 地点关联，尚未接入 M5 表达层利用家族上下文
