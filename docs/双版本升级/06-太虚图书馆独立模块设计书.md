## 一、模块定位

太虚图书馆是太虚境旗舰版的**外挂知识增强模块**，它是一个完全独立的 TypeScript 项目，与主程序共享数据规范但不共享代码依赖。

### 1.1 核心价值

| 维度 | 价值 |
|------|------|
| 知识管理 | 自动摄入外挂文件夹，将个人文档转化为可检索的语义词条 |
| 记忆增强 | 主程序高价值对话自动同步为图书馆词条，扩充知识库 |
| 双模对接 | MCP 协议对接 Obsidian 等第三方笔记工具，HTTP 服务对接主程序 |
| 规范统一 | DNA 编码、钙化分、元标签与主程序 100% 一致，数据互通 |

### 1.2 边界定义

```
包含:                    不包含:
├── 文件解析流水线          ❌ 角色判定逻辑
├── 三层知识存储            ❌ 情感分析引擎
├── MCP/HTTP 双模服务       ❌ 对话生成能力
├── 外挂文件夹监控           ❌ 用户交互界面
├── 元标签生成              ❌ 业务角色（玉瑶/秘书等）
└── 检索与同步              ❌ 人物图谱管理
```

### 3.1 DnaGenerator（DNA 生成器）

**与主程序完全一致的 DNA 编码实现**。

```typescript
// lib/taixu-library/src/core/DnaGenerator.ts

export class DnaGenerator {
  /**
   * 生成 HY-DNA 根码
   * 格式: DNA-YYYYMMDD-HHmm-NNNN-HX
   * 与主程序 M1 DNAEncoder 生成规则 100% 一致
   */
  generateRootId(userId: string, timestamp?: Date): string {
    const ts = timestamp ?? new Date();
    const dateStr = this.formatDate(ts);     // YYYYMMDD
    const timeStr = this.formatTime(ts);     // HHmm
    const seqNo = this.getNextSeqNo(dateStr); // 0000-9999
    const hyStamp = this.generateHYStamp(userId, dateStr);
    return `DNA-${dateStr}-${timeStr}-${seqNo}-${hyStamp}`;
  }

  /**
   * 生成环节特征码
   * 格式: DNA根码.模块代码.流水号
   */
  generateSubId(rootId: string, moduleCode: ModuleCode, seqNo: number = 1): string {
    return `${rootId}.${moduleCode}.${String(seqNo).padStart(3, '0')}`;
  }

  /**
   * 瑶印码生成（FNV-1a 非加密哈希）
   * 同一用户同一天产出一致
   */
  private generateHYStamp(userId: string, dateStr: string): string {
    let hash = 0x811c9dc5;
    const data = `${userId}:${dateStr}`;
    for (let i = 0; i < data.length; i++) {
      hash ^= data.charCodeAt(i);
      hash = Math.imul(hash, 0x01000193);
    }
    return (Math.abs(hash) % 16).toString(16).toUpperCase();
  }

  // ... 辅助方法
}

export type ModuleCode = 'LIB' | 'RAW' | 'WIKI' | 'IDX';
```

### 3.2 CalciumEngine（钙化分引擎）

**与主程序完全一致的钙化分计算规则**。

```typescript
// lib/taixu-library/src/core/CalciumEngine.ts

export interface CalciumInput {
  emotionalIntensity: number;  // 0-10
  intimacyLevel: number;       // 0-10
  socialRelevance: number;     // 0-10
  factualDensity: number;      // 0-10
  unusualness: number;         // 0-10
}

export class CalciumEngine {
  /** 初始钙化分计算 */
  calculate(input: CalciumInput): number {
    const score =
      input.emotionalIntensity * 0.30 +
      input.intimacyLevel * 0.25 +
      input.socialRelevance * 0.20 +
      input.factualDensity * 0.15 +
      input.unusualness * 0.10;
    return Math.round(Math.min(10, Math.max(0, score)) * 10) / 10;
  }

  /** 召回增长 */
  applyRecallIncrement(current: number): number {
    return Math.round(Math.min(10, current + 0.2) * 10) / 10;
  }

  /** 每日衰减 */
  applyDailyDecay(memory: { calcium: number; tags?: string[] }): number {
    const isWorkRelated = memory.tags?.some(
      t => ['work', 'tech', 'study'].includes(t)
    );
    const rate = isWorkRelated ? 0.05 : 0.10;
    return Math.round(Math.max(0, memory.calcium - rate) * 10) / 10;
  }

  /** 晋升判定 */
  canPromoteToWiki(calcium: number): boolean {
    return calcium >= 1.0;
  }

  canPromoteToSchema(calcium: number, recallCount: number): boolean {
    return calcium >= 4.5 || recallCount >= 5;
  }
}
```

### 3.3 EntityExtractor（实体提取）

**纯规则引擎，匹配主程序 L3EntityAnnotator 的实体提取逻辑**。

```typescript
// lib/taixu-library/src/core/EntityExtractor.ts

export interface ExtractedEntity {
  name: string;
  type: 'person' | 'place' | 'event' | 'concept' | 'object';
  confidence: number;  // 0-1
}

export class EntityExtractor {
  private rules: EntityRule[];

  constructor(rulesConfig?: EntityRule[]) {
    this.rules = rulesConfig ?? DEFAULT_RULES;
  }

  /** 从文本中提取实体 */
  extract(text: string): ExtractedEntity[] {
    const entities: ExtractedEntity[] = [];
    // FMM 分词 + 规则匹配（与主程序 L3 一致）
    for (const rule of this.rules) {
      const matches = text.match(rule.pattern);
      if (matches) {
        entities.push({
          name: matches[0],
          type: rule.type,
          confidence: rule.confidence,
        });
      }
    }
    return this.deduplicate(entities);
  }

  private deduplicate(entities: ExtractedEntity[]): ExtractedEntity[] {
    const seen = new Set<string>();
    return entities.filter(e => {
      const key = `${e.type}:${e.name}`;
      if (seen.has(key)) return false;
      seen.add(key);
      return true;
    });
  }
}

interface EntityRule {
  pattern: RegExp;
  type: ExtractedEntity['type'];
  confidence: number;
}
```

```typescript
// lib/taixu-library/src/parsers/ParserFactory.ts

export class ParserFactory {
  static getParser(filePath: string): FileParser {
    const ext = path.extname(filePath).toLowerCase();
    switch (ext) {
      case '.txt':  return new TextParser();
      case '.md':   return new TextParser();  // MD 作为增强文本
      case '.docx': return new DocxParser();
      case '.pdf':  return new PdfParser();
      case '.jpg':
      case '.jpeg':
      case '.png':
      case '.bmp':
      case '.webp': return new ImageParser();
      default:      throw new UnsupportedFormatError(ext);
    }
  }
}

export interface ParsedResult {
  content: string;         // 纯文本内容
  title?: string;          // 提取的标题
  metadata?: Record<string, string>;  // 文档元数据
  images?: string[];       // 嵌入式图片（base64）
}
```

### 4.2 解析要求

| 格式 | 解析器 | 提取内容 | 兜底策略 |
|------|--------|----------|----------|
| .txt | TextParser | 全文 UTF-8 | 非 UTF-8 自动检测编码 |
| .md  | TextParser | 纯文本 + YAML front-matter | 无 YAML 则跳过 |
| .docx | DocxParser | 段落文本 + 标题层级 + 内嵌图片 | mammoth 解析失败则报错不崩溃 |
| .pdf  | PdfParser | 逐页文本 + 元信息 | 扫描版 PDF 提示 OCR 建议 |
| .jpg/.png | ImageParser | OCR 文字 + 情感标签 | OCR 无结果则不崩溃，标记空内容 |

```typescript
    const fileName = path.basename(filePath);
    const tags = this.sceneMatcher.match(ocrResult.text, fileName);

    return {
      content: ocrResult.text,
      title: tags[0] ? `[${tags[0]}] ${fileName}` : fileName,
      metadata: { tags: tags.join(','), ocrConfidence: String(ocrResult.confidence) },
    };
  }
}
```

### 5.2 晋升规则（与主程序一致）

```typescript
export class PromotionEngine {
  constructor(
    private rawStore: RawStore,
    private wikiStore: WikiStore,
    private schemaStore: SchemaStore,
    private calciumEngine: CalciumEngine,
  ) {}

  /** Raw → Wiki 晋升（可手动触发或定时自动） */
  async promoteRawToWiki(rawDna: string): Promise<boolean> {
    const raw = await this.rawStore.getByDna(rawDna);
    if (!raw) return false;

    // 计算钙化分（原始内容的情感价值）
    const calcium = await this.calculateInitialCalcium(raw);

    if (this.calciumEngine.canPromoteToWiki(calcium)) {
      const wikiDna = this.generateDna(); // 新的 Wiki DNA
      await this.wikiStore.create({
        dna_root_id: wikiDna,
        title: this.extractTitle(raw),
        content: raw.original_content,
        calcium,
        tags: this.extractTags(raw),
        entities: this.extractEntities(raw),
        source_dna: rawDna,
      });
      return true;
    }
    return false;
  }

  /** Wiki → Schema 晋升 */
  async promoteWikiToSchema(wikiDna: string): Promise<boolean> {
    const wiki = await this.wikiStore.getByDna(wikiDna);
    if (!wiki) return false;

    if (this.calciumEngine.canPromoteToSchema(wiki.calcium, wiki.recallCount ?? 0)) {
      await this.schemaStore.create({
        dna_root_id: this.generateDna(),
        wiki_dna: wikiDna,
        keywords: this.extractKeywords(wiki),
        entities: this.mergeEntities(wiki.entities),
        topics: this.clusterTopics(wiki.tags),
      });
      await this.wikiStore.markPromoted(wikiDna);
      return true;
    }
    return false;
  }
}
```

```typescript
// lib/taixu-library/src/server/HttpServer.ts

export class HttpServer {
  private server: http.Server;
  private port: number;

  constructor(port: number = 3737, private store: LibraryStore) {
    this.port = port;
    this.server = http.createServer((req, res) => this.handleRequest(req, res));
  }

  async start(): Promise<void> {
    return new Promise(resolve => {
      this.server.listen(this.port, '127.0.0.1', () => {
        logger.info(`[Library] HTTP server listening on 127.0.0.1:${this.port}`);
        resolve();
      });
    });
  }

  async stop(): Promise<void> {
    return new Promise(resolve => this.server.close(() => resolve()));
  }

  private async handleRequest(req: http.IncomingMessage, res: http.ServerResponse) {
    // 路由分发
    const { method, url } = req;
    const route = this.resolveRoute(method!, url!);

    try {
      const result = await route.handler(req);
      this.jsonResponse(res, 200, { success: true, data: result });
    } catch (err) {
      this.errorResponse(res, err);
    }
  }
}
```

```typescript
      tags: { type: 'array', items: { type: 'string' } },
    },
    required: ['action'],
  },
};

export class McpServer {
  async handleToolCall(params: {
    action: string;
    query?: string;
    dna_root_id?: string;
    content?: string;
    tags?: string[];
  }): Promise<McpResponse> {
    switch (params.action) {
      case 'search':
        return { entries: await this.store.search(params.query) };
      case 'get':
        return { entry: await this.store.getByDna(params.dna_root_id!) };
      case 'create':
        return { entry: await this.store.create(params) };
      case 'update':
        return { entry: await this.store.update(params.dna_root_id!, params) };
      case 'delete':
        await this.store.delete(params.dna_root_id!);
        return { success: true };
      case 'sync':
        return { report: await this.syncManager.syncAll() };
      default:
        throw new Error(`Unknown action: ${params.action}`);
    }
  }
}
```

```typescript
// lib/taixu-library/src/folder-manager/FolderWatcher.ts

export class FolderWatcher {
  private watcher: fs.FSWatcher | null = null;
  private processingQueue: Set<string> = new Set();
  private debounceTimers: Map<string, NodeJS.Timeout> = new Map();

  constructor(
    private watchPath: string,
    private pipeline: ProcessingPipeline,
  ) {}

  start(): void {
    logger.info(`[Library] Watching folder: ${this.watchPath}`);
    this.watcher = fs.watch(this.watchPath, (eventType, fileName) => {
      if (!fileName || fileName.startsWith('.')) return;
      this.debounceProcess(fileName);
    });
  }

  private debounceProcess(fileName: string): void {
    const existing = this.debounceTimers.get(fileName);
    if (existing) clearTimeout(existing);

    this.debounceTimers.set(fileName, setTimeout(async () => {
      this.debounceTimers.delete(fileName);
      await this.processFile(fileName);
    }, 2000)); // 2 秒防抖，等待文件写入完成
  }

  private async processFile(fileName: string): Promise<void> {
    if (this.processingQueue.has(fileName)) return;
    this.processingQueue.add(fileName);

    try {
      const filePath = path.join(this.watchPath, fileName);
      await this.pipeline.processFile(filePath);
    } catch (err) {
      logger.error(`[Library] Failed to process ${fileName}: ${err}`);
    } finally {
      this.processingQueue.delete(fileName);
    }
  }

  stop(): void {
    if (this.watcher) this.watcher.close();
    for (const timer of this.debounceTimers.values()) clearTimeout(timer);
    this.debounceTimers.clear();
  }
}
```

### 7.3 处理流水线

```typescript
// lib/taixu-library/src/folder-manager/SyncManager.ts

export class ProcessingPipeline {
  constructor(
    private parserFactory: ParserFactory,
    private dnaGenerator: DnaGenerator,
    private metaGenerator: MetaTagGenerator,
    private rawStore: RawStore,
    private wikiStore: WikiStore,
    private archivePath: string,
  ) {}

  async processFile(filePath: string): Promise<void> {
    const fileName = path.basename(filePath);
    logger.info(`[Library] Processing: ${fileName}`);

    // 1. 计算文件哈希
    const fileHash = await this.calculateHash(filePath);

    // 2. 解析文件
    let parser: FileParser;
    try {
      parser = ParserFactory.getParser(filePath);
    } catch (err) {
      if (err instanceof UnsupportedFormatError) {
        logger.warn(`[Library] Unsupported format: ${fileName}, archiving raw`);
        await this.archiveOriginal(filePath);
        return;
      }
      throw err;
    }

    const result = await parser.parse(filePath);
    if (!result.content.trim()) {
      logger.warn(`[Library] Empty content: ${fileName}`);
      await this.archiveOriginal(filePath);
      return;
    }

    // 3. 生成 DNA 根码
    const dnaRootId = this.dnaGenerator.generateRootId('library');

    // 4. 提取实体 + 生成钙化分
    const entities = new EntityExtractor().extract(result.content);
    const calcium = new CalciumEngine().calculate({
      emotionalIntensity: 3.0,  // 文档默认中等
      intimacyLevel: 0,
      socialRelevance: entities.filter(e => e.type === 'person').length,
      factualDensity: 7.0,     // 文档通常事实密度高
      unusualness: 3.0,
    });

    // 5. 生成元标签 + MD
    const meta = this.metaGenerator.generate(
      dnaRootId, result.content, filePath, entities, calcium,
    );
    const markdown = this.metaGenerator.toMarkdown(meta, result.content);

    // 6. 写入原始附件层
    await this.rawStore.create({
      dna_root_id: dnaRootId,
      file_name: fileName,
      file_path: filePath,
      original_path: filePath,
      file_size: (await fs.stat(filePath)).size,
      mime_type: this.getMimeType(filePath),
      sha256_hash: fileHash,
      original_content: result.content,
    });

    // 7. 写入知识笔记库
    const outputPath = path.join(
      this.outputDir,
      meta.type === 'memo' ? 'memos' : 'references',
      `${dnaRootId}.md`,
    );
    await fs.writeFile(outputPath, markdown, 'utf-8');

    // 8. 写入语义词条层
    await this.wikiStore.create({
      dna_root_id: dnaRootId,
      title: meta.title,
      type: meta.type,
      content: result.content,
      summary: meta.summary,
      calcium,
      entities: JSON.stringify(entities),
      tags: JSON.stringify(meta.tags),
    });

    // 9. 归档原始文件
    await this.archiveOriginal(filePath);

    logger.info(`[Library] Successfully processed: ${fileName} → ${dnaRootId}`);
  }
}
```

### 8.1 HTTP 对接客户端（主程序侧）

```typescript
// src/app/library/LibraryClient.ts（主程序侧，新增文件）

export class LibraryClient {
  private baseUrl: string;
  private enabled: boolean;

  constructor(config: { enabled: boolean; port: number }) {
    this.enabled = config.enabled;
    this.baseUrl = `http://127.0.0.1:${config.port}`;
  }

  /** 同步高钙记忆至图书馆 */
  async syncMemory(dnaRootId: string, content: string, calcium: number): Promise<void> {
    if (!this.enabled) return;
    try {
      await fetch(`${this.baseUrl}/api/v1/entries`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          dna_root_id: dnaRootId,
          content,
          tags: ['synced-from-memory'],
        }),
      });
    } catch (err) {
      logger.warn(`[LibraryClient] Sync failed (non-blocking): ${err}`);
    }
  }

  /** 从图书馆检索知识 */
  async search(query: string): Promise<LibraryEntry[]> {
    if (!this.enabled) return [];
    try {
      const res = await fetch(`${this.baseUrl}/api/v1/entries?q=${encodeURIComponent(query)}`);
      const json = await res.json();
      return json.data ?? [];
    } catch {
      return []; // 图书馆不可用不阻塞主程序
    }
  }

  /** 健康检查 */
  async healthCheck(): Promise<boolean> {
    try {
      const res = await fetch(`${this.baseUrl}/api/v1/health`);
      return res.ok;
    } catch {
      return false;
    }
  }
}
```
