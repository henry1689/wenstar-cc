# Temporal 域（engine/temporal/）— 日历时空

**含义：** 日历时间、天气、节气、生理周期。

> ⚠️ 此 temporal = "calendar time"，不是 "memory timeline"。
> 记忆时间线的 temporal 在 `engine/tianquan/temporal/`。

模块：SolarTermEngine、weather_qweather_client、weather_time_engine、temporal_calendar

通过 SceneSnapshot.context_tags 间接关联 prefrontal/temporal。
