# /project-init — 项目初始化

## 触发条件

需要开始一个新项目时。

## 流程

### 第一步：确认项目类型
- Node.js / Python / Rust / 其他
- TypeScript / JavaScript
- 前端 / 后端 / 全栈
- 测试框架偏好

### 第二步：搭建基础结构
1. 初始化项目配置（package.json / pyproject.toml / Cargo.toml）
2. 配置 TypeScript / ESLint / Prettier
3. 配置测试框架
4. 创建目录结构
5. 初始化 git

### 第三步：创建 CLAUDE.md
记录项目基础信息、架构约定、常用命令。

### 第四步：验证
- [ ] 项目能正常启动
- [ ] 测试能跑通
- [ ] 编译无错误
